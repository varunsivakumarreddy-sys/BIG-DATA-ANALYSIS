## 📌 Overview

This project performs large-scale data analysis on an e-commerce dataset using **PySpark**, demonstrating scalability and distributed processing.

The dataset includes:

* Orders
* Customers
* Order Status
* Geo Lookup

---

## 🎯 Objective

To analyze customer behavior, order lifecycle, and regional trends using distributed data processing.

---

## 🧩 Data Schema

### Orders

* customer_id
* id (order_id)
* purchase_ts
* product_id
* product_name
* currency
* local_price
* usd_price
* purchase_platform

### Customers

* id
* marketing_channel
* account_creation_method
* country_code
* loyalty_program
* created_on

### Order_Status

* order_id
* purchase_ts
* ship_ts
* delivery_ts
* refund_ts

### Geo_Lookup

* country
* region

## 🚀 Key Analysis Performed

### 📊 Customer Analysis

* New vs returning customers
* Loyalty program impact
* Marketing channel performance

### 📦 Order Analysis

* Total revenue (USD)
* Orders per platform
* Top-selling products

### 🚚 Order Lifecycle Analysis

* Average shipping time
* Delivery delays
* Refund rate

### 🌍 Regional Insights

* Revenue by region
* Country-level performance
